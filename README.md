# Gestion de Distribution d'Examens

Application desktop développée en C++ avec Qt Creator et une base de données Oracle.

## Fonctionnalités
- Ajout, modification, suppression de distributions d'examens (CRUD complet)
- Gestion des entités Distribution et Pick-Up
- Notifications et conteur de deadlines avec buzzeur inclus
- Interface graphique intuitive avec Qt

## Technologies utilisées
- **Langage** : C++
- **Interface** : Qt Creator / Qt Framework
- **Base de données** : Oracle

## Lancer le projet
1. Ouvrir le fichier `.pro` avec Qt Creator
2. Configurer la connexion Oracle dans le code
3. Compiler et exécuter
